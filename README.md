# Waldkraiburg Bau Premium

Premium Next.js landing page for a local construction company in Waldkraiburg, Germany.

## Stack

- Next.js 15 App Router
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- shadcn/ui style components
- Lucide Icons

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
